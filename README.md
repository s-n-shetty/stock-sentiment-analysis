# Stock Sentiment Analysis

## Overview
This project analyzes textual data related to stocks to understand market sentiment and its potential influence on price movement.

## What I Did
- Cleaned and preprocessed text data
- Applied sentiment scoring techniques
- Built a classification model
- Evaluated performance metrics

## Why It Matters
Market sentiment often reflects investor behavior. Extracting insights from text data can complement quantitative financial analysis.

## Tech Used
Python, NLTK, Scikit-learn, Pandas
